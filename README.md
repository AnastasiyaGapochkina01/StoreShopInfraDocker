# M2 Docker
Зависимости
1) Docker и Docker Compose (v2+)
2) Make
3) Git
## Пошаговая установка
1) На основе env.example создать файл .env со своими значениями переменных
2) Запустить контейнеры
```
make up
```
3) Установить Magento (потребуются ключи)
```bash
make install
```
4) Настройка проекта
```bash
make setup
```

## Работа с Makefile (все доступные команды)
| Команда        | Описание                                |
| -------------- | --------------------------------------- |
| make up        | Запустить все сервисы в фоне            |
| make down      | Остановить и удалить контейнеры         |
| make restart   | Перезапустить контейнеры                |
| make build     | Пересобрать образы без кеша             |
| make logs      | Просмотреть логи контейнеров            |
| make shell-php | Подключиться к контейнеру PHP           |
| make shell-db  | Подключиться к MariaDB                  |
| make install   | Установить Magento 2 в каталог app/     |
| make setup     | Выполнить установку и настройку Magento |
| make clean     | Полностью очистить проект и данные      |
